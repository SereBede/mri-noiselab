Spatially varying signal levels#

This example simulates a synthetic magnitude image with three distinct signal levels plus background Rayleigh noise, and then applies mri_noiselab.subtract_noise() to reduce noise while preserving the different intensity regions.

Core steps in the script:

  1. Build a synthetic “true” phantom with three intensity levels concentric squares.

  2. Sample Rayleigh noise for the background (rng.rayleigh(...))

  3. Combine signal and noise in quadrature (sqrt(signal^2 + noise^2))

  4. Apply mrnl.subtract_noise(noisy_img, bg_noise, f_size=...)

  5. Compare images and print simple statistics region-wise

1) Import modules and define useful function#

 1import mri_noiselab as mrnl
 2import numpy as np
 3import matplotlib.pyplot as plt
 4
 5def img_stats (image):
 6    '''
 7    Creates a dictionary with average, standard deviation and signal to noise 
 8    ratio of a uniform region
 9
10    Parameters
11    ----------
12    image : 2D numpy array as image.
13
14    Returns
15    -------
16    stats : python dictionary where 
17        'ave': mean of the sample image,
18        'sd': standard deviation of the sample image,
19        'snr': Signal to noise ratio computed as ratio of signal and noise powers
20
21    '''
22    ave = np.mean(image)
23    sd = np.std(image)
24    
25    P_signal = np.mean(np.square(image))
26    P_noise = np.var(image)
27    
28    snr = P_signal / P_noise if P_noise != 0 else 0
29    
30    stats = {
31        'ave': ave,
32        'sd': sd,
33        'snr': snr
34    }
35    
36    return stats

2) Generate and clean 3-level image with noise#

 1## Create a 3 levels signal image affected by Rayleigh noise
 2
 3# (change these values to vary region intensities)
 4true_signal_1 = 30
 5true_signal_2 = 50
 6true_signal_3 = 100
 7# (change this value to vary noise contribution)
 8noise_level = 40
 9# (change this value to evaluate the effects of filter size)
10filter_size = 5
11
12
13# Generate a 3 levels true image with concentric squares
14true_img = np.zeros((120,120))
15true_img[20:100,20:100] = true_signal_1
16true_img[35:85,35:85] = true_signal_2
17true_img[50:70,50:70] = true_signal_3
18
19# Generate the background noise
20rng = np.random.default_rng(seed=42)
21bg_noise = rng.rayleigh(scale=noise_level/1.253, size=np.shape(true_img))
22
23# Background Noise is summed in quadrature to true signal
24noisy_img = np.sqrt(np.square(true_img) + np.square(bg_noise))
25
26## Then given bg_noise clean it in noisy_img
27cleaned_img = mrnl.subtract_noise(noisy_img, bg_noise,f_size=filter_size)
28cleaned_stats = img_stats(cleaned_img)
29

3) Compute levels statistics#

 1# Beware selecting uniform level areas for stats!
 2bg_stats = img_stats(noisy_img[0:20,:])
 3noisy_stats_1 = img_stats(noisy_img[20:35,20:100])
 4noisy_stats_2 = img_stats(noisy_img[35:50,35:85])
 5noisy_stats_3 = img_stats(noisy_img[50:70,50:70])
 6
 7
 8cleaned_stats_bg = img_stats(cleaned_img[0:20,:])
 9cleaned_stats_1 = img_stats(cleaned_img[20:35,20:100])
10cleaned_stats_2 = img_stats(cleaned_img[35:50,35:85])
11cleaned_stats_3 = img_stats(cleaned_img[50:70,50:70])
12
13
14#After cleaning the SNR should improve
15
16statistics = {'ave':'Region mean','sd':'Region std ','snr':'Region SNR '}
17
18for s in statistics.keys():
19    print(f'{statistics[s]} | Before | After \n',
20         f'Background | {bg_stats[s]:.2f} | {cleaned_stats_bg[s]:.2f}\n', 
21         f'Region 1   | {noisy_stats_1[s]:.2f} | {cleaned_stats_1[s]:.2f}\n',
22         f'Region 2   | {noisy_stats_2[s]:.2f} | {cleaned_stats_2[s]:.2f}\n',
23         f'Region 3   | {noisy_stats_3[s]:.2f} | {cleaned_stats_3[s]:.2f}\n')
24

4) Show compared results#

 1# Plot the images to see the differences
 2fig, axes = plt.subplots(1, 3, figsize=(16, 5))
 3
 4images = [true_img, noisy_img, cleaned_img]
 5titles = ["Image without noise", "Image with noise", f"After noise reduction f_size={filter_size}"]
 6
 7vmin = 0

17fig.colorbar(im, ax=(axes[0],axes[1],axes[2]), orientation='vertical',
