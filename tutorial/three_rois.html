Region-based processing#

This example performs noise subtraction one ROI at a time using masked arrays. Instead of applying mri_noiselab.subtract_noise() on the full image, it uses mri_noiselab.subtract_noise_masked() to correct only the pixels belonging to each region, then merges the corrected regions back into a final image.

Core steps in the script:

  1. Simulate a 3-level phantom and add Rayleigh background noise (magnitude model).

  2. Define disjoint ROI masks (outer ring, middle ring, core) plus a background ROI.

  3. Apply subtract_noise_masked() region-by-region (noise estimated from background).

  4. Merge cleaned regions into a single corrected image.

  5. Print statistics before vs after for each region.

0) Import modules and define helper function#

 1import numpy as np
 2import matplotlib.pyplot as plt
 3from mri_noiselab import subtract_noise_masked
 4
 5
 6def img_stats(image):
 7    """Stats over valid pixels; supports masked arrays."""
 8    if np.ma.isMaskedArray(image):
 9        data = image.compressed()
10    else:
11        data = np.ravel(image)
12
13    ave = float(np.mean(data))
14    sd = float(np.std(data))
15    p_signal = float(np.mean(np.square(data)))
16    p_noise = float(np.var(data))
17    snr = p_signal / p_noise if p_noise != 0 else 0.0
18    return {"ave": ave, "sd": sd, "snr": snr}

1) Simulate Phantom with noise#

 1# -----------------------------
 2# 1) Simulate 3-level signal + Rayleigh noise
 3# -----------------------------
 4
 5noise_level = 40
 6filter_size = 5
 7
 8# Define nested slices of concentric squares setting true signal
 9nested_slices = {
10    "Background" : ( (slice(None), slice(None)) ,  0.0),
11    "Region 1": ( (slice(20, 100), slice(20, 100)),30.0), 
12    "Region 2": ( (slice(35, 85),  slice(35, 85)), 50.0),
13    "Region 3": ( (slice(50, 70),  slice(50, 70)), 100.0),
14}
15
16true_img = np.zeros((120, 120))
17for name, ((rr, cc), true_sig) in nested_slices.items():
18    true_img[rr, cc] = true_sig
19
20rng = np.random.default_rng(seed=42)
21bg_noise = rng.rayleigh(scale=noise_level/1.253, size=true_img.shape)
22noisy_img = np.sqrt(np.square(true_img) + np.square(bg_noise))

2) Define disjoint ROIs and plot them#

 1# -----------------------------
 2# 2) Define DISJOINT region masks (one per signal level)
 3# -----------------------------
 4# Base ROIs (nested) create a dictionary 
 5roi = {}
 6for name, ((rr, cc), true_sig) in nested_slices.items():
 7    m = np.zeros_like(noisy_img, dtype=bool)
 8    m[rr, cc] = True
 9    roi[name] = m
10
11# Make them disjoint:
12# region3 = core = roi["Region 3"]
13# region2 = middle ring (roi2 minus roi3)
14roi["Region 2"] &= (~roi["Region 3"])
15# region1 = outer ring (roi1 minus roi2 or 3)
16roi["Region 1"] &= ~(roi["Region 2"] | roi["Region 3"])
17# Background = outside any other ROI
18roi["Background"] = ~(roi["Region 1"] | roi["Region 2"] | roi["Region 3"])
19
20# Plot the resulting ROIs
21fig, ax = plt.subplots(2, 2, figsize=(10, 10))
22order = [
23    ("Background", (0, 0), "Background ROI"),
24    ("Region 1",   (0, 1), "ROI Region 1 (outer ring)"),
25    ("Region 2",   (1, 0), "ROI Region 2 (middle ring)"),
26    ("Region 3",   (1, 1), "ROI Region 3 (core)"),
27]
28
29for name, (i, j), title in order:
30    ax[i, j].imshow(roi[name], cmap="gray")
31    ax[i, j].set_title(title)
32    ax[i, j].axis("off")
33
34plt.tight_layout()
35plt.show()

3) Clean one ROI at a time#

 1# -----------------------------
 2# 3) Clean one region at a time (using masked image input)
 3# -----------------------------
 4
 5# appling masks, do not forget to invert them to obtain ROIs
 6cleaned = noisy_img.copy()
 7bg_ma = np.ma.array(noisy_img, mask=~ roi["Background"])
 8
 9for name, seg in roi.items(): # seg as "segmentation" == roi
10    region_ma = np.ma.array(cleaned, mask=~seg) # (True mask => ignored pixel)
11    fill = np.mean(region_ma) 
12            
13    region_ma = subtract_noise_masked(
14        region_ma,
15        bg_ma,       # noise estimate from background
16        fill_value = fill,
17        f_size=filter_size,
18    )
19    cleaned[seg] = region_ma.data[seg]

4) Compute region-wise statistics (before vs after)#

 1# -----------------------------
 2# 4) Stats: before vs after per region
 3# -----------------------------
 4
 5statistics = {"ave": "Region MEAN", "sd": "Region STD", "snr": "Region SNR"}
 6for k, label in statistics.items():
 7    print(f"{label} | Before | After")
 8    for name, seg in roi.items():
 9        print(f"{name:10s} | {img_stats(noisy_img[seg])[k]:.2f} | {img_stats(cleaned[seg])[k]:.2f}")
10    print()

Inspecting the standard output, it is possible to observe that in the cleaned regions:

  • mean magnitude moves closer to the true signal level

  • the standard deviation is reduced

  • the SNR improves except for background, as expected
