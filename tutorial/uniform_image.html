uniform syntetic image#

Uniform image (synthetic) — Rayleigh noise + correction#

This example simulates a uniform magnitude image affected by background Rayleigh noise and then applies mri_noiselab.subtract_noise() to reduce the noise.

Core steps in the script:

  1. Build a constant “true” image (true_signal)

  2. Sample Rayleigh noise for the background (rng.rayleigh(...))

  3. Combine signal and noise in quadrature (sqrt(signal^2 + noise^2))

  4. Apply mrnl.subtract_noise(noisy_img, bg_noise)

  5. Compare images and print simple statistics

 1# -*- coding: utf-8 -*-
 2"""
 3In this example a uniform image affected by Rayleigh noise is simulated and 
 4then cleaning is performed by using subtract_noise function from mri_noiselab
 5module. In the end images are shown to see differences.
 6
 7"""
 8
 9import mri_noiselab as mrnl
10import numpy as np
11import matplotlib.pyplot as plt
12
13def img_stats (image):
14    '''
15    Creates a dictionary with average, standard deviation and signal to noise 
16    ratio of a uniform region
17
18    Parameters
19    ----------
20    image : 2D numpy array as image.
21
22    Returns
23    -------
24    stats : python dictionary where 
25        'ave': mean of the sample image,
26        'sd': standard deviation of the sample image,
27        'snr': Signal to noise ratio computed as ratio of signal and noise powers
28
29    '''
30    ave = np.mean(image)
31    sd = np.std(image)
32    
33    P_signal = np.mean(np.square(image))
34    P_noise = np.var(image)
35    
36    snr = P_signal / P_noise if P_noise != 0 else 0
37    
38    stats = {
39        'ave': ave,
40        'sd': sd,
41        'snr': snr
42    }
43    
44    return stats
45
46# Create a uniform signal image affected by Rayleigh noise
47# change those values to vary noise contribution
48true_signal = 50
49noise_level = 20
50
51
52# Generate a uniform signal image
53true_img = np.full((100,100),true_signal,dtype=float) 
54
55# Generate the background noise image
56rng = np.random.default_rng(seed=42)
57bg_noise = rng.rayleigh(scale=noise_level/1.253, size=np.shape(true_img))
58
59bg_stats = img_stats(bg_noise)
60
61# Background Noise is summed in quadrature to pure signal
62noisy_img = np.sqrt(np.square(true_img) + np.square(bg_noise))
63
64noisy_stats = img_stats(noisy_img)
65
66# Then given bg_noise clean it in noisy_img
67
68cleaned_img = mrnl.subtract_noise(noisy_img, bg_noise)
69cleaned_stats = img_stats(cleaned_img)
70
71# After cleaning the SNR should improve
72print(f'True Signal = {true_signal:.2f}, Noise Level = {noise_level:.2f} \n',
73     f'Mean intensity with Noise = {noisy_stats['ave']:.2f}, SNR = {noisy_stats['snr']:.2f},\n',
74     f'Mean intensity after noise reduction = {cleaned_stats['ave']:.2f}, SNR = {cleaned_stats['snr']:.2f}')
75
76# Plot the images to see the differences
77fig, axes = plt.subplots(1, 3, figsize=(16, 5))
78
79images = [true_img, noisy_img, cleaned_img]
80titles = ["Image without noise", "Image with noise", "After noise reduction"]
81
82vmin = 0
83vmax = max(img.max() for img in images) * 1.2
84
85# Loop all images
86for ax, img, title in zip(axes, images, titles):
87    im = ax.imshow(img, cmap='gray', vmin=vmin, vmax=vmax)
88    ax.set_title(title)
89    ax.axis('off')
90
91# Shared colorbar
92fig.colorbar(im, ax=(axes[0],axes[1],axes[2]), orientation='vertical', 
93             fraction=0.046, pad=0.04, label='Intensity')
94
95plt.show()
