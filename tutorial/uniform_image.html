Uniform signal region#

This example simulates a uniform magnitude image affected by background Rayleigh noise and then applies mri_noiselab.subtract_noise() to reduce the noise.

Core steps in the script:

  1. Build a constant “true” image (true_signal)

  2. Sample Rayleigh noise for the background (rng.rayleigh(...))

  3. Combine signal and noise in quadrature (sqrt(signal^2 + noise^2))

  4. Apply mrnl.subtract_noise(noisy_img, bg_noise)

  5. Compare images and print simple statistics

1) Import modules and define useful function#

 1import mri_noiselab as mrnl
 2import numpy as np
 3import matplotlib.pyplot as plt
 4
 5def img_stats (image):
 6    '''
 7    Creates a dictionary with average, standard deviation and signal to noise 
 8    ratio of a uniform region
 9
10    Parameters
11    ----------
12    image : 2D numpy array as image.
13
14    Returns
15    -------
16    stats : python dictionary where 
17        'ave': mean of the sample image,
18        'sd': standard deviation of the sample image,
19        'snr': Signal to noise ratio computed as ratio of signal and noise powers
20
21    '''
22    ave = np.mean(image)
23    sd = np.std(image)
24    
25    P_signal = np.mean(np.square(image))
26    P_noise = np.var(image)
27    
28    snr = P_signal / P_noise if P_noise != 0 else 0
29    
30    stats = {
31        'ave': ave,
32        'sd': sd,
33        'snr': snr
34    }
35    
36    return stats

2) Generate uniform image with noise and clean it#

 1# Create a uniform signal image affected by Rayleigh noise
 2# change those values to vary noise contribution
 3true_signal = 50
 4noise_level = 20 #intended as noise magnitude mean
 5
 6
 7# Generate a uniform signal image
 8true_img = np.full((100,100),true_signal,dtype=float) 
 9
10# Generate the background noise image
11rng = np.random.default_rng(seed=42)
12bg_noise = rng.rayleigh(scale=noise_level/1.253, size=np.shape(true_img))
13
14bg_stats = img_stats(bg_noise)
15
16# Background Noise is summed in quadrature to pure signal
17noisy_img = np.sqrt(np.square(true_img) + np.square(bg_noise))
18
19noisy_stats = img_stats(noisy_img)
20
21# Then given bg_noise clean it in noisy_img
22
23cleaned_img = mrnl.subtract_noise(noisy_img, bg_noise)
24cleaned_stats = img_stats(cleaned_img)

3) Show Results#

 1# After cleaning the SNR should improve
 2print(f'True Signal = {true_signal:.2f}, Noise Level = {noise_level:.2f} \n',
 3     f'Mean intensity with noise = {noisy_stats['ave']:.2f}, SNR = {noisy_stats['snr']:.2f},\n',
 4     f'Mean intensity after noise reduction = {cleaned_stats['ave']:.2f}, SNR = {cleaned_stats['snr']:.2f}')
 5
 6# Plot the images to see the differences
 7fig, axes = plt.subplots(1, 3, figsize=(16, 5))
 8
 9images = [true_img, noisy_img, cleaned_img]
10titles = ["Image without noise", "Image with noise", "After noise reduction"]
11
12vmin = 0
13vmax = max(img.max() for img in images) * 1.2
14
15# Loop all images
16for ax, img, title in zip(axes, images, titles):
17    im = ax.imshow(img, cmap='gray', vmin=vmin, vmax=vmax)
18    ax.set_title(title)
19    ax.axis('off')
20
21# Shared colorbar
22fig.colorbar(im, ax=(axes[0],axes[1],axes[2]), orientation='vertical', 
23             fraction=0.046, pad=0.04, label='Intensity')
24
25plt.show()

After noise reduction the mean intensity approaches the true signal and signal to noise ratio is significally improved.

True Signal = 50.00, Noise Level = 20.00 
Mean intensity with Noise = 54.64, SNR = 159.84,
Mean intensity after noise reduction = 49.97, SNR = 10298.18
