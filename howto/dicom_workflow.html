DICOM workflow#

What the script does:

  • Load a DICOM image with pydicom.dcmread(...)

  • Extract the pixel array and convert it to float for processing

  • Select a background ROI (noise-only) for Rayleigh noise estimation

  • Apply mri_noiselab.subtract_noise(...) using a local window (f_size)

  • Clip the corrected values back to the valid DICOM intensity range

  • Save a new DICOM with updated metadata (including a new SOPInstanceUID)

  • Optionally plot original vs corrected images

Requirements#

  • pydicom

  • numpy, matplotlib

  • your package (mri_noiselab)

If you haven’t yet install them:

pip install pydicom,

Background ROI advice#

Choose a region with no anatomical signal (pure background noise), and avoid:

  • any anatomy / signal spillover

  • zero-padding / masked regions

  • coil artifacts or structured background

A bad background ROI biases the Rayleigh σ estimate and can degrade the correction.

Source code#

  1# -*- coding: utf-8 -*-
  2"""
  3HOW TO:
  4Load a DICOM image, apply a background-based noise subtraction,
  5export the corrected image as a new DICOM file,
  6and visually compare original vs corrected images.
  7
  8The dicom file used in this short guide is provided in the 
  9repository, its filename is MR_femur_head.dcm
 10"""
 11
 12# --- Import required libraries ---
 13import pydicom                     # DICOM I/O
 14import numpy as np                  # Numerical operations
 15import matplotlib.pyplot as plt     # Visualization
 16from mri_noiselab import subtract_noise  # Custom noise reduction function
 17
 18
 19# ============================================================
 20# 1. Load DICOM image
 21# ============================================================
 22
 23# Read the DICOM file, containig metadata and pixel data
 24# (to open other dicom file replace with the actual file path)
 25ds = pydicom.dcmread("MR_femur_head.dcm")
 26
 27# Convert pixel data to float for numerical processing
 28# (important to avoid integer overflow during calculations)
 29img = ds.pixel_array.astype(np.float32)
 30
 31print(f"Loaded image shape: {img.shape}, datatype: {img.dtype}")
 32
 33
 34# ============================================================
 35# 2. Define background region for noise estimation
 36# ============================================================
 37
 38# Select a background area (no anatomical signal expected)
 39# This region is used to estimate noise statistics
 40background = img[0:100, 80:430]
 41# (N.B. pay attention to not include the zeros padding!)
 42
 43# ============================================================
 44# 3. Apply noise subtraction
 45# ============================================================
 46
 47# Subtract estimated noise from the image
 48# f_size controls the filter/kernel pixel size used in the algorithm
 49filter_s = 3
 50img_corr = subtract_noise(img, background, f_size=filter_s)
 51
 52
 53# ============================================================
 54# 4. Convert corrected image back to DICOM-compatible format
 55# ============================================================
 56
 57# Clip values to the valid DICOM intensity range
 58# (defined in metadata by the number of stored bits)
 59img_corr = np.clip(img_corr, 0, 2**ds.BitsStored - 1)
 60
 61# Convert back to the original integer datatype
 62img_corr = img_corr.astype(ds.pixel_array.dtype)
 63
 64
 65# ============================================================
 66# 5. Update DICOM dataset and save corrected image
 67# ============================================================
 68
 69# Replace pixel data
 70ds.PixelData = img_corr.tobytes()
 71
 72# Update image dimensions (safety check)
 73ds.Rows, ds.Columns = img_corr.shape
 74
 75# Generate a new SOP Instance UID
 76# (mandatory when modifying pixel data)
 77ds.SOPInstanceUID = pydicom.uid.generate_uid()
 78
 79# Save corrected DICOM file
 80ds.save_as("corrected_image.dcm")
 81
 82
 83# ============================================================
 84# 6. Visual comparison: original vs corrected image
 85# ============================================================
 86
 87# Create side-by-side plots
 88fig, axes = plt.subplots(1, 2, figsize=(16, 8))
 89
 90images = [img, img_corr]
 91titles = ["Original image (with noise)", f"After noise reduction (f_size={filter_s})"]
 92
 93# Shared intensity scale for fair comparison
 94vmin = 0
 95vmax = max(im.max() for im in images) * 1.2
 96
 97# Display images
 98for ax, image, title in zip(axes, images, titles):
 99    im = ax.imshow(image, cmap="gray", vmin=vmin, vmax=vmax)
100    ax.set_title(title)
101    ax.axis("off")
102
103# Add a shared colorbar
104fig.colorbar(
105    im,
106    ax=axes,
107    orientation="vertical",
108    fraction=0.046,
109    pad=0.04,
110    label="Pixel intensity"
111)
112
113plt.show()
